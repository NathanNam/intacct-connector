Class Overview

Extension of ChildDefinitionParser that treats all the child node elements as DTO that are unmarshalled with JAXB instead of the default parsing schema. This one doesn't set the attributes

Handy to use to create a DTOEvaluationInvokerMessageProcessorFactoryBean.
